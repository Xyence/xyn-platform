# Licensing Rationale

This document is project context for maintainers and contributors. It is not legal advice.

Xyn uses the GNU Affero General Public License v3.0 to keep the core platform open, require publication of modifications used over a network, permit a support and services ecosystem, and discourage closed hosted forks.

For the platform and runtime repository, that also means hosted or runtime modifications should remain visible to users interacting with the platform over a network.
